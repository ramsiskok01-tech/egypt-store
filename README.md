# Egypt Store - E-Commerce Platform

## نبذة عن المشروع
منصة تجارة إلكترونية مصرية متكاملة تجمع بين البائعين والمشترين بنظام آمن وموثوق.

## المميزات
- ✅ نظام تسجيل دخول آمن (Email, Apple)
- ✅ لوحة تحكم للمشترين والبائعين
- ✅ نظام الدفع (محفظة إلكترونية + كاش عند الاستلام)
- ✅ إدارة المنتجات مع التصنيفات
- ✅ نظام الطلبات والشحن
- ✅ لوحة تحكم الأدمن

## التثبيت

```bash
npm install
cp .env.example .env
npm start
```

## التقنيات المستخدمة
- Node.js + Express.js
- MongoDB
- React.js
- JWT Authentication
